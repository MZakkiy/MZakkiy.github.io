Pneumonia Detection from Chest X-Rays

A Deep Learning Case Study in Medical Image Analysis

Project Overview

The goal of this project was to develop a reliable and accurate deep learning model capable of classifying chest X-ray images as either 'Normal' or indicative of 'Pneumonia'. Given the critical nature of medical diagnosis, the primary objective was to achieve high sensitivity (Recall) to minimize the number of missed pneumonia cases (false negatives), providing a valuable tool for radiologists.

The dataset used in this project is the Pneumonia Dataset with Chest X-Ray dataset from Kaggle.

Python PyTorch Scikit-learn Pandas Matplotlib Seaborn

From Data to Diagnosis: The Analysis

Insight 1: The Data's Story - Imbalance and Variety

A thorough exploration of the dataset revealed two critical challenges. First, a significant class imbalance:, with far more Pneumonia images than Normal ones. This could bias a naive model. Second, the images themselves were inconsistent in size and color format.

Key Data Challenges:

  • Class Imbalance: Required a special sampling strategy during training to ensure the model learned from both classes equally.
  • Inconsistent Inputs: Necessitated a robust preprocessing pipeline to resize all images to 224x224, convert them to 3-channel RGB for the ResNet50 model, and normalize pixel values.

Insight 2: Leveraging Transfer Learning for a Powerful Model

Instead of training a model from scratch, I used transfer learning with a ResNet50 model pre-trained on ImageNet. This approach leverages a powerful, pre-existing knowledge base of visual features. The final classifier layer was replaced and trained specifically on our X-ray data, making the training process highly efficient and effective.

Insight 3: High Sensitivity - A Reliable Diagnostic Aid

The final model was evaluated on a held-out test set. The results were highly successful, particularly in the most critical metric for this use case: Recall (Sensitivity).

Key Performance Metrics:

  • Excellent Pneumonia Detection (93.6% Recall): The model correctly identified 235 out of 251 actual pneumonia cases, meaning it is very unlikely to miss a patient with the disease.
  • High Overall Accuracy (89.3%): The model was correct in its predictions nearly 90% of the time.
  • Low False Negatives: Only 16 pneumonia cases were missed. In a clinical setting, this low rate is highly desirable, making the model a reliable screening tool.

Conclusion and Future Work

This project successfully demonstrates that a well-structured deep learning pipeline can produce a highly sensitive model for medical image analysis. The ResNet50 classifier serves as a powerful proof-of-concept for an AI-assisted diagnostic tool.

Future Directions:

  • Experiment with other CNN architectures like DenseNet or EfficientNet to potentially improve performance further.
  • Deploy the model as a web application for real-time inference and demonstration.
  • Incorporate model interpretability techniques (e.g., Grad-CAM) to visualize what parts of the X-ray the model focuses on, increasing trust and transparency.
